# Zobsai Job Autofill Chrome Extension

## Overview

This Chrome Extension helps users autofill common job application fields like name, email, phone, and skills with one click.

## Features

- One-click autofill
- Uses JavaScript and DOM manipulation
- Built using Chrome Extension Manifest V3

## Installation

1. Clone the repository
2. Open chrome://extensions
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the project folder

## How to Use

1. Open any job application form
2. Click the extension icon
3. Click "Autofill Form"

## Tech Stack

- JavaScript (ES6)
- HTML & CSS
- Chrome Extensions (Manifest V3)
